# Shimmer

## Usage

```javascript
import { Shimmer } from "KnitUI"

return <Shimmer height="medium" />
```

| Prop name | Type                           | Optional | Default |
| --------- | ------------------------------ | -------- | ------- |
| height    | "large" or "medium" or "small" | Yes      | "small" |
| className | string                         | Yes      | None    |
| style     | CSS Object                     | Yes      | None    |
